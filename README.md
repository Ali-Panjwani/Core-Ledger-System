# Core-Ledger-System

### Core Ledger / Core Banking Solution for Small & Medium Financial Institutions

## Planned Features for the Project

1. Account Management 
   - Create accounts (customer, wallet, GL)
   - Balance Maintenance
   - Account status (ACTIVE, FROZEN, CLOSED)

2. Ledger & Transactions
   - Double-entry accounting (debit & credit)
   - Immutable transaction records 
   - Transaction reference & idempotency 

3. Posting Engine 
   - Validate balances 
   - Atomic transaction posting 
   - Rollback on failure

4. Balance Calculation 
   - Running balance 
   - Ledger-based balance vs cached balance

5. Audit & Traceability 
   - Who initiated a transaction 
   - When and why 
   - Complete transaction trail

6. APIs 
   - Post transaction 
   - Fetch account balance 
   - Fetch transaction history
